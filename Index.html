Welcome to Sylvester Smith's IT 4403 Project

This site is part of the coursework for IT 4403 at Kennesaw State University. It serves as an overview of my individual and term projects, which encompass several milestones focused on web development using modern technologies and APIs.

Project Milestones:

Milestone 1: JSON Integration

Involves using JavaScript and jQuery to display data from JSON files related to books. This milestone focuses on data handling and web page dynamics.

Milestone 2: Google Books API

Utilizes the Google Books API to create a book search functionality and manage a bookshelf, demonstrating API integration and client-side interaction.

Milestone 3: Books Search AJAX

Enhances the book search functionality by converting the process to AJAX, allowing dynamic content loading without reloading the page.

Milestone 4: Improved SPA UI

Improves the Single Page Application user interface and interaction, incorporating advanced features like Mustache.js for templating.

Term Project:

Term Project: Comprehensive Application

Applies all skills and knowledge developed through the individual milestones to build a comprehensive SPA, possibly incorporating a different API such as a movies API.
